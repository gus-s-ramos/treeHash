# TREEHASH


Trabalho da Faculdade


Este repositório contêm:

 A classe TableHash representa uma tabela hash personalizada e permite a inserção, recuperação e remoção de pares chave-valor. 
 
 A classe BinaryTree implementa uma árvore binária e oferece funcionalidades para inserção, recuperação, remoção e travessia da árvore. 
 
 
 A classe Node é usada internamente pela classe BinaryTree para representar os nós da árvore. 
---

Sinta-se à vontade para explorar o código e usar essas classes para implementar tabelas hash e árvores binárias em seus projetos Java. Se tiver alguma dúvida ou feedback, não hesite em entrar em contato.
